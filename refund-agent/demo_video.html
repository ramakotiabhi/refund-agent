Refund agent — project walkthrough

An AI support agent that approves, denies, or escalates refunds
against a strict, deterministic policy.

Built for e-commerce teams who need every refund decision to be consistent, explainable, and fast — at any scale, from a small store to an Amazon-sized catalog.

Every e-commerce company faces the same problem: refund requests come in by the thousands, and every one needs a consistent, defensible decision.
Architecture

One agent, not seven.

The language model's only job is to understand the customer and pick a tool. It never decides eligibility itself — that comes from deterministic code.

What's next

Built to extend — without retraining anything.

Voice is already wired as a transport into the same agent loop. Image and video evidence review is scaffolded the same way. At Amazon's scale, the pattern that matters is separating the policy from the conversation — so a policy change is a code change, not a prompt-engineering gamble across millions of requests.
